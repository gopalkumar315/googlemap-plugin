Add google map to your wordpress post and pages in less than 30 seconds with latest trending 16 styles.

== Description ==
This plugin makes it easy to work with WordPress! Add custom google map with 16 latest styles to your wordpress website pages and posts with simple easy shortcode. No iframe is used in this and very easy to use. it is very good for contact us pages.

== Installation ==

Upload the Gboy Custom Google Map Plugin Base plugin to your wordpress site and activate it. It would work as is.


== Features ==
* The most easiest Google Maps plugin on WordPress
* Easy to use, no coding required!
* Responsive maps
* Allow custom icon for you map markers
* 16 popular map themes to choose from
* Add your own map theme
* UTF-8 character support
* Allow custom height
* Choose from four Google maps types: roadmap, terrain, satellite and hybrid
* Define your own height and zoom level for your Google Map
* Create a map in less than a 30 seconds!
* No advertising or links
* Set the max zoom level for your map
* Compatible with caching plugins
* Compatible with Cloudfare
* Allow to duplicate location
* Latest Google Maps API
